loadMap

void loadMap(<String> map)
    Takes the file name of the top-left corner map and proceeds to load every other section of the map depending from information gathered from the first map. Every map has data about the maps to the top, right, bottom, and left of it, and thus this function utilizes that information, to easily load the entire map using only the name of one.
    Parameters:
      map - Map to load.
